# KN01

## A) Hypervisor Typ 1 und 2


### Begrifferklärung 
Hypervisor ist eine virtual machine monitor (VMM). Das ist eine Software die virtual machines (VMs) erstellt und läuft. 
Diese Software ermöglicht es mehreren Betriebssystemen, die Ressourcen eines einzelnen physischen Servers zu teilen.
Es verwaltet und weist Ressourcen wie CPU, Arbeitsspeicher und Speicherplatz jedem virtuellen Rechner zu, wodurch diese unabhängig laufen können und Effizienz sowie Skalierbarkeit verbessert werden.


### Unterschied zwischen Hypervisor Typ 1 und 2

#### Hypervisor 1
Ein Typ-1-Hypervisor (auch als Bare-Metal-Hypervisor bekannt, weil er auf dem Bare Metal Server sitzt) läuft direkt auf der physischen Hardware eines Computers. 


#### Hypervisor 2
Im Gegensatz zu der Typ-1-Hypervisor, ist Typ-2-Hypervisor eine Anwendung, die auf dem Host-Betriebssystem installiert ist. Er wird auch als gehosteter oder eingebetteter Hypervisor bezeichnet.

### Erklärungen/sinnvolle Überlegungen

Meine Vermutung war richtig: Mein System verwendet einen Hypervisor vom Typ 2, weil wäre es Typ 1, könnte man es "überbuchen" (overcommitment) und man hätte mehr Kontrolle über die Hardware. 

VMware Workstation läuft auf meinem Windows-Host-Betriebssystem und greift nicht direkt auf die Hardware zu, 
sondern über das Host-System. Daher kann ich keine Ressourcen über die physischen Grenzen hinaus zuweisen.

Beim Versuch, der VM mehr Prozessoren zuzuweisen, als mein Host-System hat, zeigte VMware eine Fehlermeldung.
Da VMware Workstation ein Typ-2-Hypervisor ist, läuft sie auf dem Host-Betriebssystem und hat keine direkte Kontrolle über die Hardware.
Sie kann daher keine Ressourcen überbuchen, was den Host vor Instabilität schützt.

Warum keine Fehlermeldung beim RAM?
Weil der Arbeitsspeicher im Gegensatz zu CPU-Kernen virtuell überbucht werden kann. 
VMware nutzt dabei Techniken wie Memory Ballooning oder Auslagerungsdateien (Swap), um den Speicher flexibel zu verwalten. 
Dadurch kann mehr virtueller RAM zugeteilt werden, als tatsächlich im Host-System vorhanden ist.
