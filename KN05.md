# KN05: Netzwerk / Sicherheit


## A) Grundbegriffe und private IP wählen (30%)

**VPC (Virtual Private Cloud)**
- Eine VPC ist wie das eigene private Netzwerk in der Cloud (AWS)
- Man entscheidest, welche IP-Bereiche verwendet werden, welche Subnetze existieren und wie die Instanzen miteinander kommunizieren
Vergleich: Eine VPC ist wie dein „Haus“ (alles, was drin passiert, ist privat und du kontrollierst die Türen (Firewall/Sicherheitsgruppen))

<img width="500" height="356" alt="image" src="https://github.com/user-attachments/assets/af6087d1-1cda-4284-bb0c-09f92c89177f" />



**Subnet (Subnetz)**
- Ein Subnetz ist ein Unterbereich innerhalb einer VPC, der einen Teil der IP-Adressen des VPCs enthält
- Instanzen (Server) werden einem Subnetz zugewiesen, sodass sie innerhalb dieses Bereichs kommunizieren können.
Vergleich: Wenn die VPC das Haus ist, dann ist ein Subnetz ein Raum oder Stockwerk. Jede Instanz „lebt“ in einem Raum.

**Vordefinierte Subnets**
- AWS erstellt beim Anlegen einer VPC oft automatisch Subnetze in den verfügbaren Availability Zones
- Beispiel: Eine Standard-VPC hat oft 3–6 Subnetze, verteilt über die Zonen

**Decken die Subnetze den IP-Range des VPC ab?**
Nein, nicht immer vollständig.
- Jedes Subnetz hat einen eigenen IP-Bereich, aber zusammen decken sie meist nur einen Teil des gesamten VPC-IP-Bereichs ab
- Du kannst eigene Subnetze anlegen, um den restlichen IP-Bereich zu nutzen

**Private IP**
- Wird innerhalb eines Netzwerks (VPC/Subnetz) verwendet
- Kann nur von Instanzen innerhalb desselben Netzwerks erreicht werden
Beispiel: Die interne IP eines Webservers, auf den nur der Datenbankserver zugreifen kann.

**Öffentliche IP**
- Ist im Internet erreichbar
- Ermöglicht, dass jeder von aussen auf die Instanz zugreifen kann (z. B. Browser auf Webserver)
- Wird oft automatisch vergeben, wenn eine Instanz erstellt wird.

**Statische IP (Elastic IP in AWS)**
-  bleibt immer gleich, auch wenn  Instanz gestoppt oder neu gestartet wird
-  Wichtig für Webserver, damit Clients die gleiche Adresse verwenden können
Vergleich: Wie deine feste Telefonnummer, die sich nie ändert
