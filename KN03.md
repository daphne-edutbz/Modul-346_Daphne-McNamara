**Kurze Erklärung was Ihre SQL-Abfrage (SELECT-Statement) ausliest.**

Die SQL-Abfrage:
```sql
SELECT Host, User FROM mysql.user;
```
liest aus der internen Systemtabelle mysql.user alle vorhandenen Datenbankbenutzer aus.

Dabei werden zwei Informationen angezeigt:
- User -> der Name des Datenbankbenutzers
- Host -> von welchem Host sich dieser Benutzer verbinden darf
(z. B. localhost = nur lokal, % = von überall aus erlaubt)

Damit zeigt die Abfrage also welche Benutzer existieren und von wo aus sie sich einloggen dürfen.
